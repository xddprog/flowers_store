# Памятка для заказчика

Ниже — простое описание того, что можно проверить на сайте и в админке.

## Клиентская часть (сайт)

- Просмотр каталога букетов.
- Фильтры и сортировка (по популярности и цене).
- Карточка товара с описанием и фотографиями.
- Добавление товара в корзину.
- Оформление заказа.
- Переход на оплату (открывается в новой вкладке).
- Показ только активных товаров.

## Админка

Вход по ссылке https://www.lascovo.ru/auth
логин: admin
пароль: admin

- Список товаров с пометкой «Активен/Неактивен».
- Просмотр карточки товара с фотографиями и их номерами.
- Создание товара с загрузкой нескольких изображений и превью.
- Редактирование товара:
  - изменение описания, цены, количества;
  - изменение порядка изображений;
  - удаление изображений.
- Список заказов в виде канбан‑доски:
  - перетаскивание между статусами;
  - просмотр деталей заказа.

## Оплата

- После оформления заказа открывается ссылка на оплату в новой вкладке.
